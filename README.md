PLNAME
======

A programming language name generator.

See the associated blog post
[here](https://www.ashn.dev/blog/2021-02-06-programming-language-name-generator.html).

## Use
### Script
```sh
$ ./plname.py 4
Jul
Ulqn
Soz
Xsac
```

### Module
```txt
$ python3 -q
>>> from plname import *
>>> print(plname())
Guav
```
